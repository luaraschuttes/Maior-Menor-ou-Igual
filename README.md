Maior, Menor ou Igual
Descrição do Jogo

Maior, Menor ou Igual é um jogo simples em C, onde o jogador compara dois valores numéricos e descobre qual é maior, menor ou se são iguais.

O jogo permite ao usuário escolher um atributo (como População, Área, PIB, etc.) e compara os valores das duas opções, mostrando quem venceu ou se houve empate.

Funcionalidades
Menu interativo para escolher o atributo a ser comparado.
Comparação automática dos valores do atributo selecionado.
Resultado claro indicando:
Qual opção venceu
O valor de cada opção
Empate, se houver
